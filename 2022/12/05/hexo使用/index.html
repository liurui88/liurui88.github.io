hexo使用

cnpm install -g hexo-cli 安装hexo框架

hexo -v 查看hexo版本

hexo init 生成博客 初始化博客

hexo s 启动本地服务

hexo n "XXX" 创建新的文章

hexo clean 请理

hexo g 生成

hexo d 部署到远程仓库

一般使用流程
1.hexo n “XXX”
2.hexo c
3.hexo g
4.hexo s 查看效果
5.hexo d 推送到GitHub
